# 港股美股交易规则学习笔记

## 📊 港股交易规则

### 1. 交易时间 (北京时间)
| 时段 | 时间 |
|------|------|
| 开市前时段 | 09:00 - 09:30 |
| 早市 | 09:30 - 12:00 |
| 午休 | 12:00 - 13:00 |
| 午市 | 13:00 - 16:00 |
| 收市竞价 | 16:00 - 16:10 |

### 2. 交易制度
- **T+0 交易**：当天买入，当天可以卖出
- **T+2 结算**：交易后第2个工作日结算
- **无涨跌幅限制**：股价可以无限涨跌
- **有市场波动调节机制 (VCM)**：5分钟内波动超过±10%，暂停交易5分钟

### 3. 最小交易单位
- 港股以"手"为交易单位
- **每手股数不固定**，由上市公司决定
- 常见：100股、200股、500股、1000股、2000股
- 例如：腾讯 100股/手，海底捞 1000股/手，比亚迪 500股/手

### 4. 价格档位 (Spread)
| 股价区间 (HKD) | 最小报价单位 |
|----------------|-------------|
| 0.01 - 0.25 | 0.001 |
| 0.25 - 0.50 | 0.005 |
| 0.50 - 10.00 | 0.01 |
| 10.00 - 20.00 | 0.01 |
| 20.00 - 50.00 | 0.02 |
| 50.00 - 100.00 | 0.05 |
| 100.00 - 200.00 | 0.10 |
| 200.00 - 1000.00 | 0.50 |
| 1000.00+ | 2.50 |

## 📈 美股交易规则

### 1. 交易时间 (北京时间)
| 时段 | 夏令时 | 冬令时 |
|------|--------|--------|
| 盘前 | 16:00 - 21:30 | 17:00 - 22:30 |
| 正常交易 | 21:30 - 04:00 | 22:30 - 05:00 |
| 盘后 | 04:00 - 08:00 | 05:00 - 09:00 |

### 2. 交易制度
- **T+0 交易**：当天买入，当天可以卖出
- **T+1 结算**：2024年5月28日起改为 T+1 (原来是 T+2)
- **无涨跌幅限制**：但有熔断机制

### 3. 最小交易单位
- **1股**：美股没有"手"的概念，最小交易单位是1股
- 支持碎股交易 (部分券商)

### 4. 价格档位
| 股价 | 最小变动 |
|------|----------|
| ≥$1 | $0.01 |
| <$1 | $0.0001 |

### 5. 熔断机制
#### 大盘熔断 (基于标普500)
| 触发点 | 时间 | 措施 |
|--------|------|------|
| -7% | 美东15:25前 | 暂停15分钟 |
| -13% | 美东15:25前 | 暂停15分钟 |
| -20% | 任何时间 | 全天停牌 |

#### 个股熔断 (LULD)
- 5分钟内涨跌幅超过10%
- 暂停交易5分钟
- 9:45-15:35 之间触发
